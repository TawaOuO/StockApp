# StockApp

本APP提供簡易的股票篩選<br>
．可快速依技術面、基本面、籌碼面數據篩選股票<br>
．支援多條件篩選<br>
<br>
前端使用Java開發，後端資料收集部分以python開發<br>
．串接yfinance API<br>
．使用requests,BeautyfulSoup套件收集資料<br>
．使用numpy,pandas套件整理數據<br>
<br>
原使用Web Server做每日資料更新，伺服器已關閉，目前內部僅一份資料供測試，無每日更新服務<br>
後續不在更新<br>
